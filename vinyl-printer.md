## What this machine is used for

The Roland DG TrueVIS VG3-640 is a large-format printer-cutter used for printing high-resolution graphics on roll media.

It can print full-color designs and also cut contour lines around printed graphics when the correct workflow is used.

Common uses:

- Stickers
- Decals
- Labels
- Posters
- Banners
- Exhibition graphics
- Wall graphics
- Window graphics
- Vehicle graphics
- Product labels
- Large-format signs
- Printed vinyl artwork
- GerLab branding materials
- Workshop posters and safety visuals
- Printed patterns for maker projects

This machine is useful when a project needs color graphics, large-format printing, or printed stickers with contour cutting.

## Basic Rules

- Use the vinyl printer only after instruction or staff permission.
- Use only approved print media.
- Use only approved ink and printer settings.
- Do not load unknown roll media.
- Do not touch the print head area.
- Do not touch wet ink immediately after printing.
- Check media width before printing.
- Check media alignment before printing.
- Check heater and dryer settings before printing.
- Do not pull media while the machine is printing.
- Keep hands away from moving carriage and rollers.
- Use test print before large jobs.
- Do not leave long print jobs unattended without staff approval.
- Clean the work area after use.
- Report banding, ink dropouts, media skew, head strikes, ink errors, or unusual smell.

## Safety Notes

The vinyl printer includes moving parts, ink, heaters, cutting tools, electrical parts, and heavy roll media.

Main risks:

- Moving print carriage
- Pinch rollers
- Cutting blade
- Wet ink
- Heater and dryer surfaces
- Heavy roll media
- Media skew causing head strike
- Ink or cleaning liquid contact
- Fumes or smell depending on media and ink
- Electrical risk if cables or machine are damaged

Safe behavior:

- Keep hands away while the machine is printing or cutting.
- Do not touch the blade.
- Do not open covers unnecessarily during operation.
- Do not touch the print head.
- Wear gloves when handling ink waste or cleaning liquid.
- Keep media rolls stable.
- Use two people for heavy media rolls if needed.
- Keep ventilation adequate.
- Stop the job if media wrinkles, lifts, or strikes the head.
- Follow GerLab staff instructions for cleaning and waste ink handling.

## Prohibited or Restricted Materials

Do not use unknown media.

Restricted or prohibited materials may include:

- Unknown roll media
- Dirty or dusty media
- Wet media
- Wrinkled media
- Media with damaged liner
- Media too thick for the machine
- Media too narrow or too wide for the machine
- Media that curls badly near the print head
- Heat-sensitive media not suitable for heater/dryer settings
- Media with unknown coating
- Media that releases unsafe fumes
- Hard sheet materials
- Metal sheet
- Wood
- Fabric not approved for this printer

Do not use this printer as a general-purpose flatbed printer.

## Print Resolution Notes

Good artwork resolution depends on viewing distance.

General guide:

| Use case | Suggested artwork resolution |
|---|---:|

| Posters viewed closely | 150–300 dpi at final size |

## Contour Cutting Notes

The VG3-640 can cut around printed graphics when the file is prepared correctly.

For contour cutting:

- Use vector cut lines.
- Use the correct cut contour spot color or named swatch required by VersaWorks.
- Keep cut lines clean.
- Avoid duplicate cut paths.
- Avoid tiny details that are difficult to weed.
- Leave enough bleed around the printed design.
- Let ink dry before cutting if required.
- Use appropriate blade force for the media.
- Test cut before large contour cutting jobs.

Contour cutting is useful for:

- Stickers
- Decals
- Labels
- Logo graphics
- Printed shapes
- Product graphics

## Media Loading Workflow

### 1. Check media

Before loading:

- Confirm media type.
- Confirm media width.
- Check roll direction.
- Check liner condition.
- Check for dust, wrinkles, or damage.
- Confirm the media is approved.

### 2. Load roll media

- Place roll media securely.
- Feed media through the printer.
- Align media straight.
- Position pinch rollers correctly.
- Make sure media is flat.
- Lock the media in place.

### 3. Set media

- Let the printer detect media width.
- Confirm media setup on the display.
- Check that media moves smoothly.
- Check that the leading edge is straight.

### 4. Check heater / dryer

- Select correct media profile.
- Check print heater temperature.
- Check dryer temperature.
- Allow warm-up if required.

### 5. Print test

Before a large job:

- Run test print.
- Check nozzle condition.
- Check color.
- Check banding.
- Check media feed.
- Clean or adjust if needed.

## Common Problems and Fixes

| Problem | Possible Cause | Fix |
|---|---|---|
| Banding | Nozzle issue, feed calibration, wrong media profile | Run test print, clean heads, check profile and feed |
| Missing color | Clogged nozzle, empty ink, ink flow issue | Check ink, run cleaning, ask staff |
| Ink smears | Not dry, heater too low, wrong media | Increase drying time, check heater, use correct media |
| Color looks wrong | Wrong profile, color mode issue, low-quality file | Check media profile, color settings, artwork |
| Print is blurry | Low-resolution image, wrong scale | Use higher-resolution artwork or vector file |
| Media skews | Bad loading, pinch rollers wrong, roll tension | Reload media straight, check rollers |
| Head strike | Wrinkled media, curled media, media too thick | Stop job, remove media, check approved media |
| Contour cut misaligned | Media feed issue, file setup, registration issue | Check cut path, media feed, calibration |
| Cut too shallow | Blade force too low, dull blade | Increase force, check blade |
| Cut too deep | Blade force too high | Reduce force |
| Vinyl is hard to weed | Cut settings wrong, design too detailed | Adjust force, simplify design |
| Heater does not reach temp | Environment too cold, wide media, warm-up needed | Wait for warm-up, check environment |
| Strong smell | Media or ink fumes, poor ventilation | Improve ventilation, check media approval |
| Machine error | Setup, media, ink, or maintenance issue | Read display message and ask staff |

## Maintenance Notes

Possible maintenance items:

- Run nozzle test regularly.
- Clean around print head only using approved method.
- Keep platen clean.
- Keep pinch rollers clean.
- Keep cutting area clean.
- Check blade condition.
- Check media clamps and holders.
- Check waste ink bottle.
- Keep ink system clean.
- Do not let ink dry on important parts.
- Keep machine covered or protected when not in use.
- Keep room temperature and humidity within safe range.
- Report repeated banding, head strikes, ink errors, or media feed problems.
- Do not disassemble machine without staff permission.

---

## Cleaning Notes

Cleaning must follow Roland / GerLab procedure.

General rules:

- Use only approved cleaning liquid.
- Wear gloves when handling cleaning liquid or ink waste.
- Do not touch the print head surface directly.
- Do not use random solvents.
- Do not use dirty cloth or paper that leaves fibers.
- Dispose of ink-contaminated waste according to GerLab procedure.
- Ask staff before manual cleaning.

Incorrect cleaning can damage the print head.

---

## Drying and Lamination Notes

After printing:

- Let prints dry before heavy handling.
- Let prints dry before contour cutting if needed.
- Let prints dry before lamination.
- Follow media and ink recommendations.
- Avoid rolling fresh prints too tightly.
- Keep printed surfaces clean.

Lamination may be needed for:

- Outdoor graphics
- Vehicle decals
- Floor graphics
- High-touch stickers
- Long-term signage
- Scratch resistance

---

## Application Notes

Before applying printed vinyl:

- Clean the target surface.
- Remove dust, oil, and moisture.
- Check surface texture.
- Test adhesion if needed.
- Use transfer tape for cut decals if required.
- Use squeegee carefully.
- Avoid trapping bubbles.
- Avoid applying to unsuitable surfaces.
