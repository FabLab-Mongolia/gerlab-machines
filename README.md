## Purpose

This repository helps GerLab users understand each machine before using it.

Each machine page should include:

- Machine information
- Photos
- Technical specifications
- What the machine is used for
- Basic rules
- Safety notes
- Approved materials
- Prohibited or restricted materials
- File preparation / software workflow
- Starting settings
- GerLab-tested settings
- Common problems and fixes
- Maintenance notes
- Source / manual links
