# GerLab Machines

This repository documents GerLab machines, fabrication tools, operation notes, material rules, risks, maintenance, and troubleshooting.

These pages are written for GerLab members, students, collaborators, and future makers who need clear machine information before starting work.

## Machine Areas

- [3D Printers](3d-printers.md)

- [Vacuum Former](vacuum-former.md)
- [Electronics Tools](electronics-tools.md)
- [Hand Tools and Workshop Equipment](hand-tools.md)

## What each machine page should include

Each machine guide should explain:

- What the machine is used for
- Who can use it
- Basic operation notes
- Safety risks
- Approved materials
- Prohibited materials
- File preparation
- Common problems and fixes
- Maintenance notes
- Useful settings or examples

## Goal

GerLab machine documentation should help users work confidently, safely, and creatively.

We document machines so the next maker can learn faster.
